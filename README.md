## Como Ejecutar
    

    1. en la dirrecion mern/server/ ejecutar "npm install" y "npm start"
        (*)ERROR la base de datos puede estar inactiva, avisar y la subo

    2. en la dirrecion mern/client/ ejecutar "npm install" y "npm start"
